TKVSC - Splatoon 3 Addon

An addon for TKVSC that adds support for Splatoon 3. It registers Splatoon 3 as a TKVSC game profile that allows for easy browsing and editing of game files.
This addon is in active development and is currently in beta. Please report any issues you encounter on the GitHub Issues page.
The Splatoon 3 support is largely untested, as none of our developers are knowledgable in modding Splatoon 3. If you are a Splatoon 3 modder and would like to help us test and improve this addon, please reach out to us on our Discord server.

What is TKVSC?

The Tears of the Kingdom Visual Studio Code Extension or TKVSC is a Visual Studio Code extension for editing and browsing Module System (TotK) game files.

TKVSC Website: https://tkvsc.dev/

Game Dump Guide (required): https://gamebanana.com/tuts/19858

Your Projects

The multigame project system is currently in development! TKVSC will attempt to force you to use TKMM, but you can safely delete the files relating to it.

  • Add the current folder open in VS Code to a list of Project Folders

Game Dump

  • Browse your game dump in Read-Only mode. Right click to add a file to the correct romfs path within a Project Folder of your choosing

Additional capabilities:

  • Archives within archives (e.g. a .sarc inside a .pack) can be browsed and edited
  • Search for romfs contents, including filenames inside of SARCs
  • Right-click a file within an archive to export it to a Project Folder.
  • Rather than the typical method of copying, pasting, and wiping an existing file to make a new file of the same format, TKVSC can create empty files of formats supported by the editor.
  • Preview .bntx subfiles and .txtg files as PNGs
  • For files TKVSC does not support, such as .bfres, the user can choose external programs to open them in by default (separate from similar features in the OS), or open them in the built-in hex editor.

Setup

You can find a complete setup tutorial for TKVSC on the TKVSC Docs page, or continue below.

Requirements

  • Python 3.13+
  • Microsoft C++ Redis
  • Node.js (if building extension from source)
  • Python 3.13-venv (on Linux)
  • Valid game dump

Steps

  1. Install the extension (VSIX).
  2. Follow the prompts, skipping any steps relating to game dump or TKMM.
  3. Install the addon extension.
  4. Press `Ctrl+Shift+P` to open the Command Palette and run `TKVSC Splatoon 3: Splatoon 3: Set Game Dump Path`.

Python troubleshooting

python3 works in CMD but the extension cannot find Python? VSCode is often launched without your full user PATH (unlike a terminal you opened yourself). Fix:

  1. TKVSC: Select Python (from detected installs) - scans where python3, common install folders, and py.
  2. TKVSC: Browse for python.exe - point at the real interpreter (e.g. %LocalAppData%\Programs\Python\Python314\python.exe).
  3. Or set TKVSC → Python Path to that full path, then TKVSC: Set Up Python Environment again.

If setup fails, run TKVSC: Set Up Python Environment from the Command Palette.
